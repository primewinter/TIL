### 1. 브랜치 만들기

```bash
git branch [브랜치명]
```

### 2. 브랜치 전환

```bash
git checkout [브랜치명]
git checkout -b [브랜치명]
git checkout -t [원격브랜치명]
```

- **-b :** 브랜치 작성과 전환을 동시에 하는 옵션
- **-t :** 원격 저장소의 브랜치를 영구적으로 로컬 브랜치에서 tracking하고 싶을 때

### 3. 브랜치 확인

```jsx
git branch -r : 원격저장소(깃허브) 브랜치 상태 확인
git branch -a : 로컬저장소(내 컴퓨터) 브랜치 상태 확인
```

### 4. 원격 저장소에 브랜치 생성

```bash
git push --set-upstream origin [브랜치명]
```
