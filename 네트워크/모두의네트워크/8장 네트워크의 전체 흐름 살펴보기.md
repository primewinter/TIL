# 32 랜 카드에서의 데이터 전달과 처리

## 1) 네트워크의 구성

### OSI 모델

- **응용 계층(세션 계층과 표현 계층 포함)** : ****애플리케이션 등에서 사용하는 데이터를 송수신
- **전송 계층** : 목적지에 데이터를 정확하게 전달
- **네트워크 계층** : 다른 네트워크에 있는 목적지에 데이터를 전달
- **데이터링크 계층** : 랜에서 데이터를 송수신
- **물리 계층** : 데이터를 전기 신호로 변환하는 데 필요
